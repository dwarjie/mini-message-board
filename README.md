## About

This is a simple mini-message board node application for The Odin Project.

## Tools/Language

- Nodejs
- Express
- store2

## Want to clone and customize?

You need to create a `.env` file in your root directory for the following:

```js
PORT=<local_port>
```

### Usage

1. Clone this repo

```bash
npm install
```

2. Run the command deployment

```bash
npm run start
```
